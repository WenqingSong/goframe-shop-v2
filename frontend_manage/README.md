# Goshop-backend

Based [vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)

## Project Setup

```sh
npm config set registry https://registry.npmjs.org/

npm install
```

## Compile and Hot-Reload for Development

```sh
npm run dev
```

## Compile and Minify for Production

```sh
npm run build
```
